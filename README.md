## üîß How to Use

1. **Prepare Your Files**  
   Place your Markdown files in the `markdown/NOM_COURS/` folder. 

2. **Add Your Resources**  
   Store any images or other resources in the `static/NOM_COURS/` folder.

3. **Commit and Push**  
   Save your changes by committing and pushing to your repository. The conversion will run automatically.

4. **Access Your Pages**  
   The converted pages will be available online. You can view them on GitHub Pages.

### Local Conversion

If you want to convert Markdown locally, you can use the following command in the terminal:

```bash
# Convertir tous les cours
```

Make sure you have the necessary tools installed to run this command.
